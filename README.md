# tokimo-package-jellyfin-api

Jellyfin-compatible API layer for Tokimo — auth, items, playback, sessions, and system endpoints.

## Features

| Module | Description |
|---|---|
| `auth` | API key and user authentication |
| `items` | Media library browsing and metadata |
| `playback` | Streaming and playback state |
| `session` | Active session management |
| `system` | Server info and health checks |
| `images` | Poster and backdrop serving |
| `users` | User profile management |

## Add to Cargo.toml

```toml
tokimo-package-jellyfin-api = { git = "https://github.com/tokimo-lab/tokimo-package-jellyfin-api" }
```
